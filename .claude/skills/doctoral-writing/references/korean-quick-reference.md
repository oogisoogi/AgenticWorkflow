## 논문 섹션별 문체 가이드

### 서론 (Introduction)

**목적**: 연구 배경, 문제 제기, 연구 목적 제시

```
❌ 오늘날 현대 사회에서는 매우 다양하고 복잡한 여러 가지 문제들이 발생하고 있으며, 이러한 문제들을 해결하기 위한 많은 연구들이 활발하게 진행되어 오고 있다.

✅ 디지털 전환이 가속화되면서 조직 내 세대 간 기술 격차가 심화되고 있다. 2023년 기준 국내 500인 이상 기업의 73%가 이 문제를 핵심 경영 과제로 인식한다(한국경영자총협회, 2023).
```

### 이론적 배경 (Literature Review)

**목적**: 선행연구 정리, 이론적 틀 제시

```
❌ Smith(2020)는 그의 연구에서 A가 B에 영향을 미친다고 주장하였다. 그리고 Jones(2021)도 비슷한 연구를 수행하였는데, 그 역시 A와 B 사이에 어떤 관계가 있다고 보았다. 또한 Lee(2022)의 연구에서도 이러한 관계가 확인되었다.

✅ A와 B의 관계에 관한 선행연구는 세 가지 흐름으로 구분된다. 첫째, Smith(2020)는 인과관계를 실증하였다. 둘째, Jones(2021)는 조절변수 C의 역할을 규명하였다. 셋째, Lee(2022)는 메커니즘을 질적으로 탐색하였다.
```

### 연구방법 (Methodology)

**목적**: 연구 설계, 자료 수집, 분석 방법 기술

```
❌ 본 연구에서는 설문조사 방법을 사용하여 자료를 수집하고자 하였으며, 수집된 자료는 통계 프로그램을 통해 분석되어졌다.

✅ 자료 수집은 온라인 설문으로 진행하였다. 표집 대상은 서울 소재 4년제 대학 재학생이며, 할당표집으로 320명을 선정하였다. 수집된 자료는 SPSS 26.0으로 분석하였다.
```

### 결과 (Results)

**목적**: 분석 결과 객관적 제시

```
❌ 분석 결과, 가설 1은 통계적으로 유의미하게 지지되었으며, 이는 매우 의미 있는 결과라고 할 수 있다.

✅ 가설 1의 검증 결과, A는 B에 정(+)의 영향을 미쳤다(β=.342, p<.001). 이 효과는 통제변수 투입 후에도 유지되었다(Model 2: β=.298, p<.01).
```

### 논의 (Discussion)

**목적**: 결과 해석, 이론적 함의 도출

```
❌ 이러한 연구 결과는 정말 흥미롭고 중요한 의미를 가진다고 생각된다. 왜냐하면 기존 연구에서는 이런 결과가 나오지 않았기 때문이다.

✅ 이 결과는 두 가지 측면에서 기존 연구를 확장한다. 첫째, 서구 맥락에서 도출된 이론이 한국 상황에서도 적용됨을 확인하였다. 둘째, 조절효과 분석을 통해 이론의 경계조건을 규명하였다.
```

### 결론 (Conclusion)

**목적**: 연구 요약, 시사점, 한계 및 향후 연구 제언

```
❌ 본 연구는 여러 가지 한계점을 가지고 있으며, 향후 연구에서는 이러한 한계점들을 보완하여 더욱 발전된 연구가 이루어져야 할 것으로 사료된다.

✅ 본 연구의 한계는 다음과 같다. 첫째, 횡단면 자료로 인과관계 추론에 제한이 있다. 둘째, 자기보고식 측정으로 동일방법편의 가능성이 존재한다. 후속 연구에서는 종단 설계와 다원천 자료 수집을 권고한다.
```

## 분야별 용어 사용 예시

### 사회과학

| 피해야 할 표현 | 권장 표현 |
|---------------|----------|
| 영향을 준다 | 영향을 미치다, 작용하다 |
| 관계가 있다 | 상관관계를 보인다, 연관성이 있다 |
| 차이가 있다 | 차이를 나타낸다, 유의한 차이가 있다 |
| 높아지다/낮아지다 | 증가하다/감소하다, 상승하다/하락하다 |

### 인문학

| 피해야 할 표현 | 권장 표현 |
|---------------|----------|
| ~라고 본다 | ~로 해석된다, ~로 이해할 수 있다 |
| 보여준다 | 드러내다, 현시하다, 표출하다 |
| 말하고 있다 | 논의하다, 서술하다, 기술하다 |
| 나타난다 | 발현되다, 표명되다, 현현하다 |

### 자연과학/공학

| 피해야 할 표현 | 권장 표현 |
|---------------|----------|
| 실험해 보았다 | 실험을 수행하였다, 측정하였다 |
| 결과가 나왔다 | 결과를 도출하였다, 관찰되었다 |
| 좋은 성능 | 향상된 성능, 우수한 효율 |
| 잘 작동한다 | 정상 작동한다, 설계 사양을 충족한다 |

### 1. 이중 피동

```

❌ 결과가 도출되어졌다
✅ 결과를 도출하였다
```

### 2. 주어 누락

```
❌ 분석 결과, 유의미한 차이가 나타났다.
✅ 분석 결과, 두 집단 간 유의미한 차이가 나타났다.

❌ 이론적 배경에서 살펴본 바와 같이
✅ 제2장에서 검토한 바와 같이
```

### 3. 지시어 모호성

```
❌ 이러한 결과는 이것이 중요함을 보여준다.
✅ 이 결과는 변수 A의 매개역할이 핵심적임을 시사한다.

❌ 그것은 이전 연구와 일치한다.
✅ 이 결과는 Smith(2020)의 발견과 일치한다.
```

### 4. 불필요한 완곡 표현

```
❌ ~라고 할 수 있을 것으로 생각된다
✅ ~로 판단된다

❌ ~인 것으로 사료되어진다
✅ ~이다 / ~로 해석된다
```

## 문장 변환 연습

### 원문 → 수정문 비교

**원문 1**:
> 본 연구에서는 매우 다양한 변수들 간의 복잡한 관계를 분석하고자 하였으며, 이를 통해 학문적으로 의미 있는 시사점을 도출하고자 하였다.

**수정문 1**:
> 본 연구는 변수 A, B, C 간 구조적 관계를 분석하였다. 분석 결과를 토대로 이론적·실무적 시사점을 제시한다.

---

**원문 2**:
> 설문 문항들은 기존의 선행연구들에서 타당성과 신뢰성이 검증된 측정도구들을 바탕으로 하여 본 연구의 맥락에 맞게 수정·보완되어졌다.

**수정문 2**:
> 측정도구는 Smith(2020)의 척도를 한국어로 번안하여 사용하였다. 원척도의 신뢰도는 α=.89였으며, 본 연구에서는 α=.85로 나타났다.

---

**원문 3**:
> 연구 결과, 독립변수와 종속변수 사이에서 통계적으로 유의미한 정(+)의 영향 관계가 있는 것으로 확인되었으며, 이러한 결과는 기존의 선행연구들의 결과와 일치하는 것으로 나타났다.

**수정문 3**:
> 독립변수는 종속변수에 정(+)의 영향을 미쳤다(β=.45, p<.001). 이 결과는 Jones(2021)의 연구와 일치한다.
