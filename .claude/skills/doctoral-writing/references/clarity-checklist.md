# Clarity Checklist for Doctoral Writing

학술 글쓰기의 명료성, 간결성, 학술적 엄밀성을 체계적으로 평가하고 개선하기 위한 체크리스트. 한국어와 영어 학술 원고 모두에 적용.

> **역할**: 검토용 체크리스트 (VERIFY). 원칙은 `SKILL.md`, 문제 유형별 카탈로그는 `common-issues.md`, 실전 수정 사례는 `before-after-examples.md` 참조.

## 1. 문장 수준 명료성 (Sentence-Level Clarity)

### 주어-동사 관계 (Subject-Verb Relationship)

- [ ] **주어가 명확히 식별됨** (Subject is clearly identifiable)
  - 각 문장의 주어를 3초 이내에 식별할 수 있는가?
  - 주어가 문장 시작 부분에 위치하는가?
  - Can you identify the subject of each sentence within 3 seconds?

- [ ] **주어와 동사가 가깝게 배치** (Subject and verb are close together)
  - 주어와 서술어 사이에 7-8 단어 이상의 간격이 없는가?
  - 긴 수식어를 문장 끝으로 이동하거나 제거했는가?
  - Are there fewer than 7-8 words between subject and verb?

- [ ] **동사가 강력하고 정확함** (Verb is strong and precise)
  - "이다", "있다", "하다" 등 약한 동사를 구체적 동사로 교체했는가?
  - Have you replaced weak verbs (is, has, does) with stronger alternatives?

### 문장 구조 (Sentence Structure)

- [ ] **문장 길이가 적절함** (Sentence length is appropriate)
  - 문장 길이 3단계 기준 (SKILL.md와 동일):
    - **권장**: 한국어 20-40자, 영어 15-25 words
    - **주의**: 40자/25 words 초과 시 분리 검토
    - **최대**: 60자/30 words — 반드시 분리
  - 초과 시 두 개의 명확한 문장으로 분리할 수 있는가?

- [ ] **한 문장에 하나의 주요 아이디어** (One main idea per sentence)
  - 문장이 단일 주장이나 논점에 집중하는가?
  - 복합 아이디어를 별도 문장으로 분리했는가?

- [ ] **병렬 구조 사용** (Parallel structure for related ideas)
  - 나열된 항목들이 동일한 문법 형태를 따르는가?
  - Do items in a list follow the same grammatical pattern?

### 태와 시제 (Voice and Tense)

- [ ] **능동태 적절히 사용** (Active voice used appropriately)
  - 본인의 연구 행위를 설명할 때 능동태를 사용했는가?
  - 피동태는 행위자가 불명/비중요할 때만 사용하는가?

- [ ] **일관되고 적절한 시제** (Consistent and appropriate tense)
  - 완료된 연구 행위는 과거형으로 작성했는가?
  - 일반적 진리와 기존 문헌은 현재형으로 작성했는가?

## 2. 단어 수준 정확성 (Word-Level Precision)

### 전문용어 (Terminology)

- [ ] **전문용어 첫 사용 시 정의** (Technical terms defined on first use)
  - 전문용어를 처음 사용할 때 정의를 제공했는가?
  - 정의가 대상 독자에게 명확하고 접근 가능한가?

- [ ] **약어 풀어쓰기** (Acronyms spelled out)
  - 모든 약어를 첫 사용 시 풀어서 설명했는가?
  - 형식: "Full Name (FN)" → 이후 "FN" 사용

- [ ] **일관된 용어 사용 — 용어 SOT** (Consistent terminology)
  - 동일한 개념에 대해 일관된 용어를 사용하는가?
  - 기술 용어에 대한 "우아한 변형" 지양했는가?
  - 원고 전체에서 용어가 첫 정의와 동일하게 사용되는가? (절대 기준 2)
  - 용어 일관성과 학문적 정확성이 충돌할 경우, 정확성을 우선하고 SOT 자체를 수정했는가? (절대 기준 1 > 2)

### 단어 선택 (Word Choice)

- [ ] **정확하고 구체적인 단어** (Precise and specific words)
  - 모호한 표현을 구체적인 단어로 교체했는가?
  - 사용한 단어가 의도한 바를 정확히 전달하는가?

- [ ] **불필요한 수식어 제거** (Unnecessary modifiers removed)
  - "매우", "아주", "상당히" 등 불필요한 부사를 제거했는가?
  - 남아 있는 수식어가 의미에 필수적인가?

- [ ] **전문용어 최소화** (Jargon minimized)
  - 전문 용어를 필요한 경우에만 사용하는가?
  - 정확성 손실 없이 더 명확한 대안으로 교체할 수 있는가?

## 3. 단락 수준 응집성 (Paragraph-Level Coherence)

### 단락 구조 (Paragraph Structure)

- [ ] **명확한 주제문** (Clear topic sentence)
  - 첫 문장이 단락의 주제를 명확히 제시하는가?
  - 독자가 단락의 내용을 예측할 수 있는가?

- [ ] **단락당 하나의 주요 아이디어** (One main idea per paragraph)
  - 하나의 단락이 한 가지 주제에 집중하는가?
  - 과도하게 넓은 단락을 분리했는가?

- [ ] **지지 문장들이 주제를 발전시킴** (Supporting sentences develop topic)
  - 모든 문장이 주제문을 뒷받침하는가?
  - 아이디어의 논리적 진행이 있는가?

### 전환과 흐름 (Transitions and Flow)

- [ ] **문장 간 효과적 전환** (Effective transitions between sentences)
  - 문장 간 관계가 명시적인가?
  - 적절한 전환 단어/구문을 사용했는가?

- [ ] **단락 간 명확한 연결** (Clear connections between paragraphs)
  - 각 단락이 이전 단락과 논리적으로 연결되는가?
  - 주요 아이디어 간 전환이 자연스러운가?

- [ ] **논리적 논증 전개** (Logical argument progression)
  - 아이디어가 체계적으로 쌓이는가?
  - 전체 논증 구조가 명확한가?

## 4. 간결성 (Conciseness)

### 장황함 제거 (Wordiness Reduction)

- [ ] **중복 표현 제거** (Redundancies eliminated)
  - "과거의 역사", "최종 결과" 같은 중복 표현을 제거했는가?
  - "past history", "end result" 등 영어 중복 표현도 확인했는가?

- [ ] **장황한 구문 교체** (Wordy phrases replaced)
  - "~에 있어서" → 제거 또는 "~에서"
  - "~에 대하여" → "~을/를" 또는 제거
  - "in order to" → "to"
  - "due to the fact that" → "because"

- [ ] **과도한 전치사구/조사 축소** (Excessive prepositional phrases reduced)
  - "의"가 연속적으로 3개 이상 나오지 않는가? (한국어)
  - Can you replace chains of prepositional phrases with simpler structures?

### 명사화 (Nominalization)

- [ ] **불필요한 명사화 전환** (Unnecessary nominalizations converted)
  - "~을/를 실시하다" → 구체적 동사 사용
  - "conduct an investigation" → "investigate"
  - "make a decision" → "decide"

### 피해야 할 구문 (Constructions to Avoid)

- [ ] **"~이다/~가 있다" 구문 최소화**
  - "~라는 것이 중요하다" → 직접 서술
  - "There are three factors that..." → "Three factors..."

- [ ] **헤징을 적절한 수준으로 축소** (Hedging reduced to appropriate levels)
  - "아마도", "어쩌면", "~할 수도 있다"를 과도하게 사용하지 않는가?
  - 정확성을 위한 헤징인가, 습관적 헤징인가?

## 5. 학술적 엄격성 (Academic Rigor)

### 인용과 증거 (Citations and Evidence)

- [ ] **주장이 증거로 뒷받침됨** (Claims supported with evidence)
  - 각 주장이 적절한 인용이나 증거를 갖추고 있는가?
  - 출처가 최신이고 권위 있는가?

- [ ] **인용이 올바르게 형식화됨 — 인용 SOT** (Citations properly formatted)
  - 인용 형식이 요구되는 양식(APA, Chicago 등)을 따르는가?
  - 모든 본문 내 인용이 참고문헌 목록에 포함되는가?
  - 원고 전체에서 단일 인용 스타일이 일관 적용되는가? (절대 기준 2)

### 학술적 어조 (Scholarly Tone)

- [ ] **적절한 격식 유지** (Appropriate formality maintained)
  - 일관되게 격식 있는 학술적 어조를 유지하는가?
  - 축약형과 구어체를 피했는가?

- [ ] **객관적 언어 사용** (Objective language used)
  - 주장을 감정적이 아닌 객관적으로 제시하는가?
  - 과장이나 주관적 판단을 피했는가?

- [ ] **학문 분야에 적절한 인칭/태 — 스타일 SOT** (Appropriate person/voice for discipline)
  - 분야 관례에 따라 1인칭/3인칭을 사용했는가?
  - 서술 목소리가 원고 전체에서 일관적인가? (절대 기준 2)
  - 일관성을 위해 특정 섹션의 논증 품질이 저하되는 경우, 해당 섹션에서 예외를 허용하고 사유를 명시했는가? (절대 기준 1 > 2)

## 6. 분야별 관례 (Discipline-Specific Conventions)

- [ ] **요구되는 스타일 가이드 준수** (Required style guide followed)
  - APA, Chicago, MLA, 기타 요구 형식?
  - 형식 규칙이 일관되게 적용되는가?

- [ ] **학문적 규범 준수** (Disciplinary norms respected)
  - 태(능동/피동), 인칭(1인칭/3인칭), 시제, 섹션 구성 관례를 따르는가?

## How to Use This Checklist

### 전체 원고 리뷰 시
1. 각 섹션에서 대표적 2-3개 단락에 체크리스트 적용
2. 오류 패턴 파악 (예: 일관된 피동태 남용)
3. 문서 전체에 체계적으로 교정 적용
4. 수정 후 우선순위 높은 섹션 재점검

### 집중 수정 시
1. 특정 문제 식별 (예: 장황함, 불분명한 주어)
2. 관련 체크리스트 섹션에 집중
3. 교정 적용
4. 체크리스트로 개선 확인

### 자기 편집 시
1. 초안 작성 후 24-48시간 대기 후 검토
2. 초안을 소리 내어 읽어 어색한 표현 포착
3. 체크리스트를 체계적으로 적용
4. 수정 후 반복

### 우선순위

**High Priority** (의미와 명료성에 영향):
- 주어-동사 관계
- 전문용어 정의
- 문장/단락당 하나의 주요 아이디어
- 주장의 근거 뒷받침

**Medium Priority** (가독성에 영향):
- 문장 길이
- 능동/피동태 균형
- 전환 표현
- 장황함 제거

**Low Priority** (완성도에 영향):
- 형식 일관성
- 사소한 문체 변형
- 선택적 헤징 축소
